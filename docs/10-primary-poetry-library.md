# 小学诗词馆开发说明

日期：2026-07-16

## 功能定位

诗词馆是识字系统里的“答后文化语境与自由阅读”模块，不是第二套背诵打卡产品。它通过一首完整诗、一个可理解的画面和少量考据说明，让孩子在不影响快速识字主流程的前提下，看见单字如何进入词句、景物和文化记忆。

## 数据与覆盖

- 总量：75 篇；汉 2、北朝 1、唐 46、宋 20、元 1、明 1、清 4。
- 每篇字段：标题、作者、朝代、完整诗文、儿童释义、情绪、历史边界、画面依据、证据等级、图片和无障碍替代文本。
- 图片：75 幅；其中 8 幅位于 `public/images/classics/`，67 幅位于 `public/images/poems/`。
- 运行时来源：`src/data/primary-poems.ts`。
- 提示词清单生成器：`scripts/poem-image-manifest.ts`。

## 界面

- 主导航与首页均提供“诗词馆”入口；
- 可搜索诗名、作者、朝代、诗句、释义和情绪；
- 可按汉魏北朝、唐、宋、元明清筛选；
- 默认显示 12 篇，再按 12 篇展开，图片使用 `loading=lazy`；
- 详情展示完整诗文、儿童释义、诗的气质、时代与考据边界、构图理由；
- 自动抽取诗中常用汉字作为“认一认”提示，并明确不写入掌握度；
- 关闭弹窗支持背景点击、关闭按钮和 Escape 键。

## 不干扰识字的约束

1. 不在孩子独立回想汉字之前出现诗词提示；
2. 不把打开、朗读或浏览诗词作为识字正确证据；
3. 不设置连续打卡、排行或强制背诵任务；
4. 不用大模型在线向儿童自由对话；
5. 图片与扩展说明可跳过，核心扫描和复习离线可用。

## 验收

- 数据测试确认 75 个连续 ID、75 个唯一 slug、每篇完整字段与合法证据等级；
- 资源测试确认所有 75 个图片路径真实存在；
- 67 幅新图统一为 1280×853 JPEG，总量约 25MB；
- PWA 图片缓存上限 80 条，经典图和诗词馆图共用 `poetry-scenes-v2` 缓存；
- 桌面和手机断点均可使用，手机端诗词详情改为单列；
- `npm test` 与 `npm run build` 必须同时通过后才可部署。
