# 古诗时代情境图考据与制作规范

日期：2026-07-16

## 1. 产品结论

古诗配图有价值，但价值来自“帮助孩子看懂诗中的动作、空间、季节和比喻”，不是用漂亮古风图增加装饰。图片只出现在答后的文化彩蛋中，默认收起，不进入计时、评分、掌握度和复习排程；因此它不会打断单字独立回忆。

系统把画面分为三个证据等级：

- **史实较明确**：作品、时代和主要地点或媒介关系较清楚，细节仍是复原；
- **部分可考**：时代或大背景清楚，但写作地点、文本版本或流传故事存在争议；
- **情境复原**：只能根据诗句和同时代生活资料构造一个可能场景。

所有图片统一标注“原创情境复原图 · 不是历史照片或作者肖像”。在没有可靠图像证据时，人物采用匿名、远景或背影，不能把生成面貌说成李白、孟浩然等真实容貌。

## 2. 资料方法与视觉边界

服饰、生活器物和建筑首先参考博物馆及专业文化机构资料，不以电视剧、游戏立绘或现代景区复建作为一手依据：

- [中国国家博物馆：中国古代服饰文化展](https://www.chnmuseum.cn/zl/ztcl/202102/t20210205_248954.shtml)及其[数字展览](https://www.chnmuseum.cn/portals/0/web/zt/202102gdfsh/)用于判断唐代男子幞头、圆领袍、靴及宋代衣着趋于质朴收敛等时代特征；
- [中国国家博物馆：唐代男子陶俑](https://www.chnmuseum.cn/zp/zpml/kgfjp/202111/t20211116_252268.shtml)用于核对唐代普通人物轮廓，不复制文物形象；
- [敦煌研究院：敦煌唐代服饰文化](https://www.dha.ac.cn/clzl/lnzl/yyns__dhtdfswhjcxsjxsz.htm)及[壁画中的社会生活](https://www.dha.ac.cn/info/1021/5490.htm)用于交叉核对服饰、交通和生活场景；
- [故宫博物院：《清明上河图》的北宋城市图景研究](https://www.dpm.org.cn/journal_detail/239215.html)用于理解北宋建筑与社会生活，不能把画中具体人物或店铺搬到另一首诗里；
- [故宫博物院：耕织图专题](https://www.dpm.org.cn/ancient/hall/161234.html)用于了解耕织题材的图像传统；现存具体作品年代与诗作不同时，只能作为方法参照，不能直接当作唐代农具证据。

工作顺序固定为：核对作者/作品归属 → 核对朝代和可能地点 → 区分实景、回忆、比喻、夸张或题画诗 → 选择同时代视觉资料 → 写清不可确认项 → 生成原创草图 → 人工检查时代错置、现代物件、伪文字和人物肖像暗示 → 通过后才发布。

## 3. 20 首诗的考据边界

| 作品 | 等级 | 画面处理边界 |
|---|---|---|
| 《江南》 | 情境复原 | 汉乐府，作者和地点不可考；只复原江南采莲生活。 |
| 《咏鹅》 | 部分可考 | 传统署名骆宾王；“七岁作诗”属后世传述，不画儿童诗人肖像。 |
| 《风》 | 情境复原 | 无具体地点；秋叶与二月花是两季并置，不伪装成同一瞬间。 |
| 《咏柳》 | 情境复原 | 只根据早春柳枝与丝带比喻构图，不指定真实河岸。 |
| 《登鹳雀楼》 | 史实较明确 | 鹳雀楼、黄河关系明确；不用今天的复建楼替代唐代楼阁。 |
| 《春晓》 | 情境复原 | 具体住所不可考；采用朴素唐代居室和鸟鸣意象。 |
| 《静夜思》 | 部分可考 | 文本版本与写作地点有争议；只画通用旅舍和月光。 |
| 《古朗月行》 | 情境复原 | 是童年想象的文学表达；匿名儿童只用于解释“白玉盘”比喻。 |
| 《望庐山瀑布（其二）》 | 史实较明确 | 庐山明确；“三千尺”是夸张，不按数字机械还原。 |
| 《绝句》 | 部分可考 | 常联系成都草堂时期；鸟、柳、青天比作者面貌可靠。 |
| 《赋得古原草送别》 | 情境复原 | 不采用未经可靠证明的应试轶事，只画古原草木。 |
| 《山行》 | 情境复原 | 具体山岭说法不一，不把现代景区当写作现场。 |
| 《江雪》 | 部分可考 | 常联系永州时期，但雪江是凝练文学场景；舟中人不作为柳宗元肖像。 |
| 《悯农（其二）》 | 部分可考 | 传统署名李绅；农人劳动不等于作者现场目击。 |
| 《小池》 | 情境复原 | 具体池塘不必强定，只复原泉眼、树阴和柔光。 |
| 《题西林壁》 | 史实较明确 | 苏轼 1084 年游庐山背景较清楚；不臆造当年题壁原貌。 |
| 《惠崇春江晚景》 | 史实较明确 | 这是题画诗，应呈现“画中之景”，不能画成苏轼河岸写生。 |
| 《春日》 | 部分可考 | 当时泗水地区的历史条件复杂；采用象征性春景，不宣称实地踏青。 |
| 《宿新市徐公店》 | 部分可考 | 题目提供地点线索，店舍原貌和儿童身份不可考。 |
| 《忆江南》 | 史实较明确 | 是离开江南后的回忆；画面应带记忆感，不画作者现场写生。 |

上述边界已经进入结构化数据，孩子或家长可以在卡片内展开查看，而不是只留在开发文档里。

## 4. 首批发布与图像提示词

首批选择最能借助画面解释动作或比喻的 8 首：《江南》《咏鹅》《风》《咏柳》《登鹳雀楼》《春晓》《静夜思》《古朗月行》。图片保存在 `public/images/classics/`，采用 3:2 横幅、1280 像素长边 JPEG；不写文字、不含水印，避免模型生成伪汉字。浏览器只在孩子展开相关卡片时加载，并采用独立运行时缓存，不增加核心字库的预缓存体积。

统一提示词骨架如下，每首再替换场景事实和禁用项：

> 为中国小学识字应用创作一幅 3:2 横向、儿童友好的历史情境插画。依据〔朝代与可确认场景〕，突出〔帮助理解诗意的动作/空间/比喻〕；服饰、发式、家具、器物和建筑参考同时代材料，人物匿名、面貌不清晰，不塑造作者肖像。矿物颜料与温和水墨结合，轮廓清楚、情绪宁静、适合 7—10 岁儿童。不要现代物件，不要后世服饰，不要现代景区复建，不要文字、题款、水印或边框。

每幅还必须加入诗作自己的反误导约束。例如《静夜思》不得指定城市，《风》明确采用两季并置，《登鹳雀楼》不得复制现代复建建筑，《古朗月行》不得把匿名儿童称为幼年李白。

## 5. 后续验收门槛

其余 12 首不会自动批量补齐。每幅图必须同时通过内容审核（是否确实帮助理解）、史料审核（是否越过可确认边界）、儿童界面审核（是否喧宾夺主）和技术审核（替代文本、尺寸、延迟加载、缓存）。如果某首诗无法形成比文字说明更清楚的画面，就保留考据卡而不配图。
